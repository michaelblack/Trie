Trie
====

An implemetation of a Trie in haskell

The sample program `main.hs` loads words from a file and gives the first 5 autocompletions for the words you type.

Example :

```
bash$ ./main /usr/local/share/dict/words
>>= imperv
imperverse
impervertible
impervestigable
imperviability
imperviable
>>= const
constable
constablery
constableship
constabless
constablewick
>>= constan
constancy
constant
constantan
constantly
constantness
>>= ^C
```
